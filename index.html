固件更新

PICO摇杆芯片已经安装了最新的GP2040固件，如果有新的固件，你需要按以下操作：

1. 从 https://github.com/FeralAI/GP2040/releases 下载最新固件；

2. 按住PICO上的白色BOOTSEL按钮并插入PC电脑；

3. 连接PC电脑时，电脑中会出现一个新的磁盘，将"*.uf2"文件直接拖进这个磁盘；

4. 更新完成后PICO将会自动重启并识别成摇杆控制器；

模式切换

在插入摇杆时按住以下按钮开机将会切换模式，切换模式掉电也能保存。

1P - PS3/Dinput：适用于PS3/4主机模式

1K - Nintendo Switch：适用于任天堂Switch

2K - PC/XInput：适用于PC电脑或XBOX

D-Pad模式

切换D-Pad模式，在开机按以下按钮：

SELECT + START + DOWN - 正常D-Pad模式

SELECT + START + LEFT - 左模拟摇杆模式

SELECT + START + RIGHT - 右模拟摇杆模式

SOCD模式

切换SOCD模式，在开机按以下按钮：

HOME + START + DOWN - 回中 (U+D=N, L+R=N)

HOME + START + UP - 方向上优先(U+D=U, L+R=N)

HOME + START + LEFT - 最后输入优先

RGB灯光

支持WS2812 RGB灯光特效，在激活后按以下键操作：

SELECT + START + 1P - 下一个特效

SELECT + START + 1K - 上一个特效

SELECT + START + 2P - 增加亮度

SELECT + START + 2K - 降低亮度

SELECT + START + 3P - LED Parameter Up

SELECT + START + 3K - LED Parameter Down

SELECT + START + 4P - Pressed Parameter Up

SELECT + START + 4K - Pressed Parameter Down

高级配置

PICO内置了一个配置应用程序，需要从PC电脑访问程序，按以下操作：

1. 在插入摇杆时按住START开机；

2. 打开浏览器输入 http://192.168.7.1；

在这个界面中可以调整摇杆的设置，配置RGB LED，重置摇杆等操作；

连发模式（V1.2新增）

为1P/2P/3P/4P/1K/2K/3K/4K 8个按键增加了连击模式 ：

连击模式每秒可以在 1 到 60 次之间进行调整；

如果连接了OLED，可以看到当前连击速度和TMRBO设置的按钮数量

开启/关闭连发模式 - 按住TURBO(FN1)按钮和要设置连发的按钮

调整连发的速度 - 按住TURBO + UP 提高速度，按住TURBO + DOWN 降低速度

V1.0版本的用户也可以使用连发键：PGM BUTTON中的3(连发)、4(地)引脚为连发键，下图红色数字区域；
